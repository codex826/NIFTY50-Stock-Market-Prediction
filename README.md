# NIFTY50-Stock-Market-Prediction
NIFTY 50 Stock Prediction - 350 Character Description: Stock prediction app using LSTM to forecast NIFTY 50 prices. Built with Streamlit &amp; TensorFlow. Upload CSV data, train 3-layer LSTM model, view interactive charts. Predicts 7-60 days ahead. Customizable lookback, epochs &amp; batch size. Perfect for traders, investors &amp; students learning ML.
